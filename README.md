Proyek ini merupakan aplikasi berbasis web yang dirancang untuk membantu pengguna memantau tingkat stres harian berdasarkan kebiasaan tidur, aktivitas harian, dan kondisi emosi. Sistem menghitung indeks kondisi mental secara sederhana dan mengelompokkan hasil ke dalam kategori stres rendah, sedang, atau tinggi.

Aplikasi ini ditujukan sebagai alat bantu pemantauan mandiri yang mudah digunakan, tidak memerlukan instalasi khusus, dan dapat diakses melalui perangkat desktop maupun mobile.

Fitur Utama • Input jam tidur harian • Pemilihan jenis aktivitas. Ringan, sedang, atau berat • Input durasi dan catatan aktivitas • Penentuan tingkat stres otomatis • Grafik tingkat stres harian dengan sumbu waktu dan skala numerik • Indikator warna tingkat stres. Hijau, kuning, merah • Perhitungan rata rata stres mingguan • Saran otomatis berdasarkan kondisi stres • Penyimpanan data permanen menggunakan localStorage • Fitur reset data tanpa pop up

Teknologi yang Digunakan • HTML sebagai struktur halaman • CSS untuk desain antarmuka yang responsif dan ramah perangkat mobile • JavaScript untuk logika perhitungan, pengolahan data, dan visualisasi grafik

Cara Menjalankan Aplikasi Unduh atau clone repository ini Buka file index.html menggunakan browser Isi data harian sesuai kondisi Lihat hasil analisis, grafik, dan riwayat stres

Struktur Folder • index.html • style.css • script.js

Tujuan Pengembangan Memberikan solusi sederhana untuk pemantauan tingkat stres berbasis aktivitas sehari hari serta meningkatkan kesadaran pengguna terhadap pentingnya keseimbangan antara istirahat dan aktivitas.

CatatanAplikasi ini bersifat edukatif dan tidak menggantikan diagnosis profesional di bidang kesehatan mental.
